# Nurney Plant and Civil Ltd Website

## Overview
A static business website for Nurney Plant and Civil Ltd, a company specializing in plant machinery hire and civil engineering services based in Nurney, Co Carlow, Ireland.

## Project Structure
- `index.html` - Main homepage
- `PrivacyPolicy.html` - Privacy policy page
- `sitemap.xml` - SEO sitemap
- `assets/` - Static assets including:
  - `images/` - Company images and logos
  - `bootstrap/` - Bootstrap CSS framework
  - `theme/` - Custom theme styles
  - `mobirise/` - Mobirise builder assets

## Tech Stack
- Static HTML/CSS/JS
- Bootstrap for responsive layout
- Mobirise website builder

## Development
Run the web server workflow which serves the site using Python's HTTP server on port 5000.

## Deployment
Configured for static deployment - all files in the root directory are served directly.
